Godot 4.5 橫向卷軸遊戲課程專案 README
========================================

【一、課程目標】
本課程將使用 Godot 4.5 製作一款橫向卷軸遊戲。
完成後，你將擁有一個可遊玩的完整作品，並可匯出成執行檔帶回家。

本課程重點：
- 學習基本 Godot 操作
- 使用模組組裝遊戲
- 不需具備程式基礎
- 以完成作品為主要目標


【二、專案資料夾結構說明】

Project/
│
├─ Assets/          美術素材（圖片、音效）
│
├─ Modules/         功能模組（已完成程式）
│   ├─ Player/
│   ├─ Enemy/
│   ├─ Coin/
│   ├─ HUD/
│   └─ Manager/
│
├─ Levels/          關卡場景
│
├─ Steps/           課程階段備份
│
├─ Demo/            完整範例專案
│
└─ main.tscn        主場景


【三、主要模組說明】

1. Player（角色模組）
   - 控制角色移動與跳躍
   - 可調整速度與跳躍高度

2. Enemy（敵人模組）
   - 自動移動
   - 碰撞判定與傷害

3. Coin（收集物模組）
   - 碰到後消失
   - 增加分數

4. HUD（介面模組）
   - 顯示生命與分數

5. Manager（系統管理模組）
   - 控制關卡切換
   - 管理遊戲狀態


【四、課堂基本操作流程】

每一堂課請依照以下流程進行：

Step 1：開啟專案
- 開啟 Godot
- 點選 Import
- 選擇本專案資料夾

Step 2：開啟主場景
- 開啟 main.tscn
- 確認畫面正常顯示

Step 3：加入模組
- 從 FileSystem 拖曳模組至場景
- 放置在適當位置

Step 4：調整參數
- 點選節點
- 在 Inspector 中修改數值

Step 5：測試遊戲
- 按 F5 或 Play
- 確認功能是否正常


【五、關卡製作流程】

1. 建立新場景
   - 新增 Node2D 或 TileMap
   - 存到 Levels 資料夾

2. 建立地形
   - 使用 TileMap 放置地板
   - 確認可行走區域

3. 放置物件
   - 放入 Player
   - 放入 Coin
   - 放入 Enemy

4. 設定鏡頭
   - 加入 Camera2D
   - 設定跟隨角色

5. 儲存並測試


【六、程式使用原則】

本課程以「組裝模組」為主，不要求自行撰寫完整程式。

請遵守以下原則：

- 不隨意修改模組內核心程式
- 只修改老師說明過的部分
- 不確定時先詢問再動手
- 出錯時先還原備份


【七、備份與還原】

每個階段皆有備份資料：

Steps/
├─ step1.zip
├─ step2.zip
├─ step3.zip

如專案發生問題：

1. 解壓對應步驟檔案
2. 重新匯入 Godot
3. 繼續課程


【八、匯出遊戲流程（最後一堂課）】

1. 開啟 Export 設定
   Project → Export

2. 新增平台
   新增 Windows / Mac / Linux

3. 設定主場景
   設為 main.tscn

4. 點選 Export
   輸出執行檔

完成後即可分享遊戲給他人遊玩。


【九、常見問題】

Q：畫面沒有顯示角色？
A：確認 Player 是否加入場景，並設為主角色。

Q：角色無法移動？
A：確認是否使用正確模組，並重新執行。

Q：程式出錯怎麼辦？
A：先回復最近備份，再請老師協助。


【十、學習建議】
